The next vowel is 으. This vowel is noteable because it doesn't have a y-vowel version.

으

으 is an important vowel. While this is a common sound in English, it’s even more common in Korean. This is the vowel sound you make in words like “foot”, “put”, and “would”. You’ll find this vowel also shows up frequently when Koreans write English words in Hangul. This is because 으 is the Korean version of the schwa1, like a neutral vowel. It’s the default sound they use when they are filling spaces between consonants in English words. Take the word “hot dog” for instance. Koreans borrowed this word from English and render it as 핫도그. In English it’s only two syllables but the constraints of 한글 means the word is three syllables (we now can write the name for the Korean alphabet, Hangul, in Korean!).

으 does go into compound-vowel, but we need to learn one more vowel first. For now go ahead with the practice below.
